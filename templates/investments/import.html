The one-time historical catch-up: build ONE CSV with your entire investment career — every contribution, withdrawal, move between accounts, dated value point, and account closure — and load it here. Download the template (it includes a sample CD-to-brokerage history). After the catch-up, ongoing money in/out comes from your checking imports (Ledger → Sync), and moves/snapshots get entered on the Ledger.

CSV columns

date — YYYY-MM-DD · account — must match an account name exactly (create missing/closed-years accounts on the Overview first) · type — contribution / withdrawal / move / snapshot / dividend / interest / fee / gain_loss / close · amount — positive dollars (all types except snapshot/close) · to_account — moves (required) and close (optional; blank = cashed out) · value — snapshots only: the account's absolute value that day · note — optional.
Rows are processed in date order. Within one account keep it chronological: contribute before you snapshot, snapshot before you move/close. Import is all-or-nothing — any bad row rolls the whole file back with the row number.
