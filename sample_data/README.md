# Sample Data

Three synthetic bank statements for trying Personal Financial Tracker without using your own money data: `sample_checking.csv` (Chase-checking format), `sample_creditcard.csv` (Chase credit-card format), and `sample_venmo.csv` (Venmo statement format).

Every row is fake — invented vendors, invented people, invented amounts. Nothing here comes from a real account.

To try them: log in, go to Tools → Import → CSV Upload, import each file against the matching starter account (My Checking / My Credit Card / Venmo), then work the Review Queue. Full walkthrough: docs/handbook.html, section "Try it with sample data".
